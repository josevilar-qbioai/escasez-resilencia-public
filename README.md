## The Model

$$V(t) = \text{Capital} \times (1+r)^t \times \frac{\Phi_L(t)}{\Phi_L(0)}$$

$$\Phi_L(t) = 1 + \frac{K}{1 + e^{-\gamma(t - t_0)}}$$

The logistic curve $\Phi_L(t)$ — Verhulst (1838) — captures the diffusion of technological self-replication (AI + robotics). Normalisation by $\Phi_L(0)$ guarantees $V(0) = \text{Capital}$ regardless of the parameters chosen.

**Parameters:**

| Parameter | Description | Calibration |
|-----------|-------------|-------------|
| `r` | Base compound return | SOX trailing 12M / 100 |
| `K` | Maximum technology multiplier | Wave amplitude |
| `γ` | Adoption speed | LLM release cadence / fab cycles |
| `t₀` | Inflection year (peak acceleration) | SOX/Copper detector |

### Generate the model curves

```bash
python scripts/plot_curves.py
# Saves: figuras/modelo_curvas_comparativa.png
```

### Backtesting with your own data

`backtest.py` expects price CSVs in the following format:

```
fecha,precio_cierre,fuente,notas
2022-01-03,43.21,yahoo,
2022-01-04,43.85,yahoo,
```

```bash
python scripts/backtest.py --hist-dir /path/to/your/csvs --out-dir results/
```

Personal portfolio data **is not included** in this repository.

## Interactive Notebook

`notebooks/Laboratorio_Modelo_Escasez_Resiliencia.ipynb` is self-contained — no external data required. It includes:

- Mathematical foundations of the logistic model
- Visualisation of $\Phi_L(t)$ by scenario (Base / Accelerated / Optimal)
- $V(t)$ projections and sensitivity analysis for $K$, $\gamma$, $t_0$
- Model properties: multiplier boundedness, limit behaviour, comparison with exponential models
- Robustness analysis of the SOX/Copper detector as an empirical estimator of $t_0$

## Portfolio Taxonomy

The paper describes a portfolio built on four scarcity pillars:

| Pillar | Assets | Rationale |
|--------|--------|-----------|
| **Digital scarcity** | Bitcoin | Fixed supply by protocol |
| **Physical scarcity** | Copper, Precious metals | Inelastic geological resources |
| **Energy / Grid** | Uranium, Smart Grid | AI infrastructure bottleneck |
| **AI self-replication** | AI, Robotics, Quantum ETFs | Direct exposure to $K$ and $\gamma$ |
| **Resilience** | MSCI World, index funds | Volatility buffer |
